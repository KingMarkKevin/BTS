BTS 1.1

  1. A cosa serve bootstrap?
    • Bootstrap è un framework front-end gratuito per uno sviluppo Web più rapido e semplice
    • Bootstrap include modelli di progettazione basati su HTML e CSS per tipografia, moduli, pulsanti, tabelle, navigazione, modali, caroselli di immagini e molti altri, nonché plug-in JavaScript opzionali
    • Bootstrap ti dà anche la possibilità di creare facilmente progetti reattivi
  2. Cosa devi importare per creare una pagina web con bootstrap?
    3. <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
  3. Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

    4. Il CDN (Content Delivery Network) serve per far collaborare più computer in una rete internet. Un vantaggio del CDN è il caricamento della cache quando si visita il sito e di conseguenza tempi di caricamento più rapidi

  4. A cosa serve questo attributo del tag meta?
    5. <content="width=device-width, initial-scale=1>
    • La width=device-width imposta la larghezza della pagina per seguire la larghezza dello schermo del dispositivo.
    • La initial-scale=1 imposta il livello di zoom iniziale quando la pagina viene caricata per la prima volta dal browser.
  5. Che differenza c'è tra Container e Container Fluid?
    • La classe ".container" fornisce un contenitore responsive a larghezza fissa.
    • La classe ".container-fluid" fornisce un contenitore a larghezza intera , che copre l'intera larghezza della finestra.
